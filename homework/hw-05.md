## 3. 回归模型设定

请分别估计以下两个模型，并将估计结果保存为 `M1` 和 `M2`：

* **模型 1 (M1)**：白人样本

  ```stata
  reg lnwage age grade married south exper tenure if race == 1
  ```

* **模型 2 (M2)**：黑人样本

  ```stata
  reg lnwage age grade married south exper tenure if race == 2
  ```

> 注意：请先生成对数化工资变量
>
> ```stata
> gen lnwage = ln(wage)
> ```
